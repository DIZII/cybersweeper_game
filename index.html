    Quick help

    Protocols

    Runner alias
    Enter a name before starting a cipher run so clean runs can be ranked.
    New cipher
    Starts a fresh randomized cipher for the selected difficulty.
    Daily cipher
    Starts today's deterministic cipher for the selected difficulty.
    Safe ping
    Uses one ping charge to highlight a safe hidden node.
    Mark mode
    Turns normal clicks into mark clicks: flag, question mark, clear.
    Custom field
    Builds a cipher from your rows, columns, and errors. Errors are capped at 70% of the field.
    Scale
    Changes cell size for larger visibility or denser oversized boards.
    Leaderboard
    Stores the fastest clean runs locally for the current mode.
    Music
    Starts or pauses the soundtrack, switches tracks, and keeps volume gentle by default.
